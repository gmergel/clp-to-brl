# Conversor CLP para BRL

Aplicacao Angular mobile-first para conversao rapida de pesos chilenos (CLP) para reais (BRL), com tema escuro e historico de conversoes.

## Rodar localmente

```bash
npm.cmd install
npm.cmd start
```

Abra em http://localhost:4200/

## Build de producao

```bash
npm.cmd run build:prod
```

## Deploy em GitHub Pages

Este repositorio esta configurado para publicar em:

https://gmergel.github.io/clp-to-brl/

Comando de deploy:

```bash
npm.cmd run deploy
```

O deploy publica a branch gh-pages automaticamente usando base href /clp-to-brl/.
